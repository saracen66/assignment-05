Important FAQ

What is the Document Object Module (DOM)?

The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

How do you select an element from DOM

To select an element from the DOM in JavaScript, you can use several methods. The most common is document.getElementById(), which selects an element by its unique ID. For selecting elements based on other attributes, you can use document.querySelector() to select the first element that matches a CSS selector or document.querySelectorAll() to select all matching elements. Additionally, document.getElementsByClassName() selects all elements with a specific class, and document.getElementsByTagName() selects all elements with a particular tag name. Each of these methods allows you to manipulate or interact with specific elements on the page.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element, allowing it to handle events triggered by its child elements. Instead of adding individual event listeners to multiple child elements, the parent captures events as they "bubble up" from the children through the DOM hierarchy. This is useful because it improves performance, especially when dealing with many child elements, and simplifies code management by reducing the number of event listeners. It also allows you to handle dynamically added child elements without needing to reattach listeners.

How do you manipulate an element's attributes and style using the DOM?

To manipulate an element's attributes in the DOM, you can use methods like setAttribute() to set an attribute or access properties like className or id directly. For styling, use the style property to apply inline CSS, such as element.style.color = 'blue'. You can also manage classes with classList.add(), classList.remove(), or classList.toggle() for more flexible control over an element's styling. These techniques allow for dynamic changes to both attributes and styles of elements on a page.
